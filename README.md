# Тестовые задания (SQL / Python / ML)

В данном репозитории собраны решения тестовых заданий, охватывающих три направления:

1. **SQL-аналитика**
2. **Python-аналитика**
3. **Python ML-моделирование**

Всего представлено **9 задач**, расположенных по возрастанию сложности:

**Junior → Junior+ → начинающий Middle**

---

## 📂 Структура репозитория

Репозиторий содержит решения задач, разбитые по направлениям:

- `SQL` — SQL-запросы и пояснения
- `Python` — аналитические задачи на Python
- `ML` — задачи машинного обучения

Каждая задача решена независимо и включает необходимые комментарии.

---

## 🧠 Что включают решения

### SQL-задачи
- SQL-запрос(ы)
- Краткое описание логики решения

### Python-задачи
- Код решения
- Пояснение ключевых шагов

### ML-задачи
- Код модели
- Используемые метрики
- Краткие выводы по результатам

---

## ⚙️ Технические допущения

- **SQL-диалект:** предпочтительно `Oracle SQL`  
  *(допускается `PostgreSQL`, если логика сохраняется)*

- **Python:** версия `3.10+`

- **Разрешенные библиотеки:**
  - `pandas`
  - `numpy`
  - `scikit-learn`
  - `scipy`
  - `xgboost / lightgbm / catboost`
